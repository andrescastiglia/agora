# Consentimiento para el piloto de Agora

Este formulario documenta el consentimiento individual para participar en el
grupo cerrado del piloto de Agora. Se completa una copia por participante y se
conserva fuera de Git junto con la evidencia de aceptación.

## Información para la persona participante

Agora recibe los mensajes y documentos compartidos en el grupo oficial del
piloto, conserva el contenido mientras el proyecto esté activo o hasta una
solicitud válida de eliminación, y lo usa para construir una base de
conocimiento y responder preguntas invocadas con `@agora`.

Meta procesa la mensajería de WhatsApp. OpenAI recibe el contenido necesario
para generar embeddings y respuestas. Los documentos originales se almacenan
como binarios en PostgreSQL, dentro de la infraestructura administrada por el
responsable. Las condiciones completas están en:

- https://agora.maese.com.ar/privacy
- https://agora.maese.com.ar/terms
- https://agora.maese.com.ar/data-deletion

La participación es voluntaria. La persona puede solicitar acceso, exportación
o eliminación escribiendo a `acastiglia@gmail.com`; la identidad se verificará
antes de ejecutar la solicitud.
